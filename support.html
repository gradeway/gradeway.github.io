Known Issues

Please Read First:

GradeWay is only a reflection of your HAC website. If you encounter an error viewing grades for example, you should first check if your grades are available on your district's HAC website before reporting the issue.

This section contains known problems that are affecting some users of the latest version of the app, and are in the process of being fixed and updated. Please make sure you are using the latest verion of the app by checking for updates in the App Store or Play Store.

What If Calculator Gray Box

A long gray box appears when you attempt to add a grade using the what if calculator. An update with a fix for this issue will be released by 1/28/2021.

If you have encountered other problems with the app, please contact me at gradeway.official@gmail.com.
